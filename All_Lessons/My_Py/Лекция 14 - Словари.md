# –°–ª–æ–≤–∞—Ä–∏ –≤ Python

<p align="justify"><font color="#7f7f7f">–°—Ç–æ–∏—Ç —Å—Ä–∞–∑—É —Å–∫–∞–∑–∞—Ç—å –æ —Ç–æ–º, —á—Ç–æ —Å–ª–æ–≤–∞—Ä—å –≤ Python - —ç—Ç–æ —Ö—ç—à —Ç–∞–±–ª–∏—Ü–∞. –û–¥–Ω–∞–∫–æ –µ–µ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ –º—ã —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –Ω–µ –±—É–¥–µ–º, –¥–∞ –∏ –≤ –ø—Ä–∏–Ω—Ü–∏–ø–µ –≥–æ–≤–æ—Ä–∏—Ç—å –æ —Ö—ç—à —Ç–∞–±–ª–∏—Ü–∞—Ö —Ç–æ–∂–µ. –ú—ã –±—É–¥–µ–º –∑–∞—Ç—Ä–∞–≥–∏–≤–∞—Ç—å –ª–∏—à—å –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫—É—é —á–∞—Å—Ç—å –¥–∞–Ω–Ω–æ–≥–æ —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö.</font> </p>

<p align="justify">–≠—Ç–∞ —Ç–µ–º–∞ —Å–ª–æ–∂–Ω–æ–≤–∞—Ç–∞ –¥–ª—è –Ω–æ–≤–∏—á–∫–æ–≤ –ø–æ—ç—Ç–æ–º—É –Ω–µ —Ä–∞—Å—Å—Ç—Ä–∞–∏–≤–∞–π—Ç–µ—Å—å –µ—Å–ª–∏ –Ω–µ –ø–æ–Ω—è–ª–∏ —á—Ç–æ-—Ç–æ —Å –ø–µ—Ä–≤–æ–≥–æ —Ä–∞–∑–∞. </p>

<p align="justify"><font color="#00b050">–°–ª–æ–≤–∞—Ä—å</font> (dictionary) ‚Äî —ç—Ç–æ –∫–æ–ª–ª–µ–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Å–æ—Å—Ç–æ–∏—Ç –∏–∑ –ø–∞—Ä "–∫–ª—é—á-–∑–Ω–∞—á–µ–Ω–∏–µ". –≠—Ç–æ –∏–∑–º–µ–Ω—è–µ–º—ã–π –∏ —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö (–Ω–∞—á–∏–Ω–∞—è —Å Python 3.7).</p> 

–û—Å–Ω–æ–≤–Ω–∞—è —Ñ–∏—à–∫–∞ —Å–ª–æ–≤–∞—Ä—è - —ç—Ç–æ –±—ã—Å—Ç—Ä—ã–π –¥–æ—Å—Ç—É–ø –∫ –¥–∞–Ω–Ω—ã–º.

<p align="justify">–°–ª–æ–≤–∞—Ä—å –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —Å –ø–æ–º–æ—â—å—é —Ñ–∏–≥—É—Ä–Ω—ã—Ö —Å–∫–æ–±–æ–∫ `{}`, —É–∫–∞–∑–∞–≤ –ø–∞—Ä—ã -- –∫–ª—é—á: –∑–Ω–∞—á–µ–Ω–∏–µ. </p>

<p align="justify"><font color="#e36c09">–ö–ª—é—á–∏</font> (key) –≤ —Å–ª–æ–≤–∞—Ä–µ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω—ã–º–∏ –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å—Ç—Ä–æ–∫–∏, —á–∏—Å–ª–∞, –∫–æ—Ä—Ç–µ–∂–∏).</p>

<p align="justify"><font color="#00b0f0">–ó–Ω–∞—á–µ–Ω–∏—è</font> (value) –º–æ–≥—É—Ç –±—ã—Ç—å –ª—é–±—ã–º–∏ –æ–±—ä–µ–∫—Ç–∞–º–∏, –≤ —Ç–æ–º —á–∏—Å–ª–µ –∏–∑–º–µ–Ω—è–µ–º—ã–º–∏.</p>

```python
# –ø—É—Å—Ç–æ–π —Å–ª–æ–≤–∞—Ä–∏–∫

# —Ç–∞–∫ —á–∞—â–µ –≤—Å–µ–≥–æ —Å–æ–∑–¥–∞—é—Ç. –≠—Ç–æ —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω—ã–π –ø–æ–¥—Ö–æ–¥
```

<p align="justify"><font color="#7f7f7f">–ó–∞–º–µ—Ç—å—Ç–µ -- –º–Ω–æ–∂–µ—Å—Ç–≤–æ –Ω–µ–ª—å–∑—è –±—ã–ª–æ —Å–æ–∑–¥–∞—Ç—å –ø—Ä–æ—Å—Ç—ã–º–∏ —Ñ–∏–≥—É—Ä–Ω—ã–º–∏ —Å–∫–æ–±–∫–∞–º–∏, –ø–æ—Ç–æ–º—É —á—Ç–æ —Ç–∞–∫ —Å–æ–∑–¥–∞–µ—Ç—Å—è —Å–ª–æ–≤–∞—Ä—å.</font> </p>

<p align="justify">–°–ª–æ–≤–∞—Ä—å —è–≤–ª—è–µ—Ç—Å—è –∏–∑–º–µ–Ω—è–µ–º—ã–º –æ–±—ä–µ–∫—Ç–æ–º, —á—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –≤—ã –º–æ–∂–µ—Ç–µ –∏–∑–º–µ–Ω—è—Ç—å –µ–≥–æ —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ –ø–æ—Å–ª–µ —Å–æ–∑–¥–∞–Ω–∏—è.</p>


<p align="justify">–ö–∞–∂–¥—ã–π –∫–ª—é—á –≤ —Å–ª–æ–≤–∞—Ä–µ —É–Ω–∏–∫–∞–ª–µ–Ω. –ï—Å–ª–∏ –≤—ã –ø–æ–ø—ã—Ç–∞–µ—Ç–µ—Å—å –¥–æ–±–∞–≤–∏—Ç—å –Ω–æ–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç —Å —É–∂–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–º –∫–ª—é—á–æ–º, —Å—Ç–∞—Ä–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è —ç—Ç–æ–≥–æ –∫–ª—é—á–∞ –±—É–¥–µ—Ç –∑–∞–º–µ–Ω–µ–Ω–æ –Ω–æ–≤—ã–º.</p>

<br>

–î–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏—è –ø–æ –∫–ª—é—á—É –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Å–∏–Ω—Ç–∞–∫—Å–∏—Å `dict[key]`. –ï—Å–ª–∏ –∫–ª—é—á —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –≤ —Å–ª–æ–≤–∞—Ä–µ, —Ç–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è –µ–≥–æ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ. –ï—Å–ª–∏ –∫–ª—é—á–∞ –Ω–µ—Ç, —Ç–æ –±—É–¥–µ—Ç –≤—ã–∑–≤–∞–Ω–∞ –æ—à–∏–±–∫–∞ `KeyError`.


# key() value() items()

<p align="justify">–ú–µ—Ç–æ–¥ <font color="#e36c09">keys()</font> –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –≤—Å–µ—Ö –∫–ª—é—á–µ–π –≤ —Å–ª–æ–≤–∞—Ä–µ. –≠—Ç–æ –Ω–µ —Å–ø–∏—Å–æ–∫, –∞ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π –æ–±—ä–µ–∫—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è, –∫–æ—Ç–æ—Ä—ã–π –º–æ–∂–Ω–æ –∏—Ç–µ—Ä–∏—Ä–æ–≤–∞—Ç—å, –Ω–æ –µ—Å–ª–∏ –Ω—É–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å –∏–º–µ–Ω–Ω–æ —Å–ø–∏—Å–æ–∫, –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é list().</p>
<p align="justify">keys() –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –∏—Ç–µ—Ä–∞—Ü–∏–∏. –≠—Ç–æ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –∫–æ–ø–∏–µ–π —Å–ø–∏—Å–∫–∞, –∞ —Å—Å—ã–ª–∫–æ–π –Ω–∞ –∫–ª—é—á–∏ –≤ —Å–ª–æ–≤–∞—Ä–µ.</p>
