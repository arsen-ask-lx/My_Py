# –†–∞—Å–ø–∞–∫–æ–≤–∫–∞ `*args`  `*kwargs`

<p align="justify">–í Python <font color="#00b050">*args</font> –∏ <font color="#00b0f0">**kwargs</font> ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–±—ã –ø–µ—Ä–µ–¥–∞—á–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–≥–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –≤ —Ñ—É–Ω–∫—Ü–∏—é.</p>

<br>

<p align="justify"><font color="#00b050">*args</font> –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–≥–æ —á–∏—Å–ª–∞ –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ –≤ —Ñ—É–Ω–∫—Ü–∏—é. –≠—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –ª—é–±–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤, –∏ –æ–Ω–∏ –±—É–¥—É—Ç —Å–æ–±—Ä–∞–Ω—ã –≤ –∫–æ—Ä—Ç–µ–∂.</p>

```python
def sum_numbers(*args):
    return sum(args)

print(sum_numbers(1, 2, 3))  # 6
print(sum_numbers(10, 20, 30, 40))  # 100
```

<br>
<p align="justify"><font color="#00b0f0">**kwargs</font> –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø–µ—Ä–µ–¥–∞—á–∏ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–≥–æ —á–∏—Å–ª–∞ –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã—Ö –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤. –≠—Ç–∏ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Å–æ–±–∏—Ä–∞—é—Ç—Å—è –≤ —Å–ª–æ–≤–∞—Ä—å, –≥–¥–µ –∫–ª—é—á–∞–º–∏ —è–≤–ª—è—é—Ç—Å—è –∏–º–µ–Ω–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤, –∞ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏ ‚Äî –∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è.</p>

```python
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30, city="New York")

# –í—ã–≤–æ–¥
name: Alice
age: 30
city: New York
```

<br>

–ú–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∏ `*args`, –∏ `**kwargs` –≤ –æ–¥–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ —Å–Ω–∞—á–∞–ª–∞ –∏–¥—É—Ç –ø–æ–∑–∏—Ü–∏–æ–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã, –∑–∞—Ç–µ–º ‚Äî –∏–º–µ–Ω–æ–≤–∞–Ω–Ω—ã–µ.

```python
def greet(x, y, *args, **kwargs):
    print(x, y)
    print("Positional arguments:", args)
    print("Keyword arguments:", kwargs)

greet("Hello", "Alice", "Bob", age=25, city="London")

# –í—ã–≤–æ–¥
Hello Alice
Positional arguments: ('Bob',)
Keyword arguments: {'age': 25, 'city': 'London'}
```

<br>

```python
def example(a, b, *args, c=10, **kwargs):
```
