# Copy –∏ deepcopy

<p align="justify">–í Python —Å—É—â–µ—Å—Ç–≤—É—é—Ç –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Å–ø–æ—Å–æ–±–∞ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏—è –æ–±—ä–µ–∫—Ç–æ–≤: <font color="#00b050">–ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ</font> (shallow copy) –∏ <font color="#00b0f0">–≥–ª—É–±–æ–∫–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ</font> (deep copy). –û–Ω–∏ –æ—Ç–ª–∏—á–∞—é—Ç—Å—è —Ç–µ–º, –∫–∞–∫ —Ä–∞–±–æ—Ç–∞—é—Ç —Å –≤–ª–æ–∂–µ–Ω–Ω—ã–º–∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞–º–∏ –¥–∞–Ω–Ω—ã—Ö. –î–∞–≤–∞–π—Ç–µ —Ä–∞–∑–±–µ—Ä–µ–º –≤—Å—ë –ø–æ–¥—Ä–æ–±–Ω–æ, —Å –ø—Ä–∏–º–µ—Ä–∞–º–∏, —Ö–∏—Ç—Ä–æ—Å—Ç—è–º–∏ –∏ —Ç–æ–Ω–∫–æ—Å—Ç—è–º–∏.</p>

<font color="#00b050">–ü–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ</font>

<p align="justify">–°–æ–∑–¥–∞–µ—Ç—Å—è –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç, –Ω–æ –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–ø–∏—Å–∫–∏ –≤–Ω—É—Ç—Ä–∏ —Å–ø–∏—Å–∫–∞) <font color="#ff0000">–Ω–µ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è</font>. –í–º–µ—Å—Ç–æ —ç—Ç–æ–≥–æ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è —Å—Å—ã–ª–∫–∏ –Ω–∞ –Ω–∏—Ö.</p>

–î–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –º–µ—Ç–æ–¥ copy() –∏–∑ –º–æ–¥—É–ª—è copy() 

```python
import copy

print("–û—Ä–∏–≥–∏–Ω–∞–ª:", original)  # [1, 2, [99, 4]]
print("–ö–æ–ø–∏—è:", shallow_copy)  # [1, 2, [99, 4]]

# –í–ª–æ–∂–µ–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ `[3, 4]` –Ω–µ –±—ã–ª —Å–∫–æ–ø–∏—Ä–æ–≤–∞–Ω
```


–î–ª—è –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç–Ω–æ–≥–æ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏—è —Å–ø–∏—Å–∫–æ–≤ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å—Ä–µ–∑—ã:

```python

```

---

<font color="#00b0f0">–ì–ª—É–±–æ–∫–æ–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ</font>

–°–æ–∑–¥–∞–µ—Ç –Ω–æ–≤—ã–π –æ–±—ä–µ–∫—Ç, –∏ –≤—Å–µ –≤–ª–æ–∂–µ–Ω–Ω—ã–µ –æ–±—ä–µ–∫—Ç—ã —Ç–∞–∫–∂–µ –∫–æ–ø–∏—Ä—É—é—Ç—Å—è.
